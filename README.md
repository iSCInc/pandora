# Pandora

An API proxy service for productizing the Mercury API with the
[guidelines](https://github.com/Wikia/guidelines/tree/master/APIDesign).

There is a draft design document [here](DESIGN.md).

## Usage

At the REPL:

```clojure
; launch the server in dev
user=> (start)
...
user=> (stop)
```
